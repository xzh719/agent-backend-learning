后端基础 & 网络协议

HTTP、WebSocket、SSE —— Agent 通信的基石

2-3 周 ▼

HTTP 协议深入

  • 请求/响应模型、状态码、Headers
  • RESTful API 设计原则
  • Content-Type、CORS、缓存策略
  • 实践：用 FastAPI / Express 构建 REST API

WebSocket 实时通信

  • WS 握手、帧格式、心跳机制
  • 房间管理、广播、点对点消息
  • 对比：WebSocket vs HTTP vs SSE
  • 实践：用 ws / socket.io 实现聊天

SSE 服务端推送

  • Server-Sent Events 协议细节
  • text/event-stream、自动重连
  • Agent 中的流式输出场景
  • 实践：实现 LLM token 流式返回

异步编程模型

  • Python asyncio / Node.js Event Loop
  • 协程、Future、Task 生命周期
  • 并发 vs 并行、GIL 的影响
  • 实践：高并发 API 服务

认证与安全基础

  • JWT、OAuth 2.0 授权码流程
  • API Key 管理最佳实践
  • HTTPS/TLS 原理
  • 实践：实现 API Key + JWT 双认证

Agent 核心架构

3 周 ▼

Tool / Function Calling

  • JSON Schema 工具定义规范
  • 并行工具调用 (parallel_tool_calls)
  • 工具结果注入对话上下文
  • 实践：实现天气/搜索/计算器工具

ReAct / 思考链模式

规划与任务分解

MCP 协议 (Model Context Protocol)

  • MCP 架构：Client-Server 模型
  • Tool/Resource/Prompt 三大原语
  • stdio / HTTP + SSE 传输
  • 实践：编写 MCP Server + Client

多 Agent 协作

  • Orchestrator-Worker 模式
  • Agent 间消息传递协议
  • 任务委派与结果汇总
  • 实践：主 Agent 调度子 Agent

# AI Agent 后端学习 —— 详细教程

> 基于 roadmap.html，逐个知识点深入展开，理论与实践结合。

## 目录结构

```
agent-backend-learning/
├── 01-http-websocket-sse/        # 阶段1：后端基础 & 网络协议
├── 02-llm-api-integration/       # 阶段2：LLM基础 & API集成
├── 03-agent-core-architecture/   # 阶段3：Agent核心架构
├── 04-memory-rag/                # 阶段4：记忆 & 知识检索
├── 05-tool-ecosystem-security/   # 阶段5：工具生态 & 安全沙箱
├── 06-observability-evaluation/  # 阶段6：可观测性 & 评估
└── 07-deployment-production/     # 阶段7：部署 & 生产化
```

## 学习方式

每个文件夹包含该阶段的多个 `.md` 教程文件，每个文件 = 一个知识点，结构如下：

1. **概念讲解** — 是什么、为什么重要
2. **核心原理** — 深入机制层
3. **代码实践** — 可运行的示例代码
4. **Agent 场景** — 在 Agent 后端中的具体应用
5. **常见问题** — 踩坑与解决方案
